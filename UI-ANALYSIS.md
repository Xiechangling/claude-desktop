## 🎨 整体布局结构

### 三栏布局
```
┌─────────────┬──────────────────────────┬─────────────┐
│             │                          │             │
│  左侧边栏   │      主内容区域          │  右侧边栏   │
│  (Sessions) │      (Messages)          │  (Context)  │
│             │                          │             │
│   60-80px   │        flex-1            │   280-320px │
└─────────────┴──────────────────────────┴─────────────┘
```

### 左侧边栏 (Session List)
- **宽度**: ~60-80px（窄边栏，仅图标）
- **背景色**: 深色 (#1a1a1a 或类似)
- **内容**:
  - 顶部: 模式切换按钮（Code/Cowork/Chat）
  - 中间: Session 列表（垂直排列的图标）
  - 底部: 设置按钮

### 主内容区域 (Message Area)
- **背景色**: 中等深色 (#252525 或类似)
- **布局**:
  ```
  ┌─────────────────────────────────┐
  │  Session Header                 │  ← 顶部固定
  │  (标题 + 操作按钮)              │
  ├─────────────────────────────────┤
  │                                 │
  │  Messages List                  │  ← 可滚动区域
  │  (对话消息)                     │
  │                                 │
  ├─────────────────────────────────┤
  │  Context Selector               │  ← 输入框上方
  │  (Local, app, main, worktree)   │
  ├─────────────────────────────────┤
  │  Input Area                     │  ← 底部固定
  │  (输入框 + Auto accept)         │
  └─────────────────────────────────┘
  ```

### 右侧边栏 (Context Panel)
- **宽度**: ~280-320px
- **背景色**: 与主内容区相同或略浅
- **内容**:
  - 文件树
  - 选中的上下文
  - 可折叠/展开

## 🎯 关键 UI 组件分析

### 1. Session Header
**位置**: 主内容区顶部

**结构**:
```
┌─────────────────────────────────────────────┐
│  📝 Session Title              [⋯] [×]      │
│  Created: 2024-04-15 19:30                  │
└─────────────────────────────────────────────┘
```

**样式**:
- 高度: ~60-80px
- 边框: 底部有细线分隔 (border-bottom)
- 标题字体: 16-18px, 加粗
- 时间戳: 12-14px, 灰色
- 操作按钮: 右对齐，图标按钮

---

### 2. Context Selector (关键差异！)
**位置**: 输入框正上方（不是在 session header 下方）

**官方设计**:
```
┌─────────────────────────────────────────────┐
│  ☑ Local    ☐ app    ☐ main    ☑ worktree  │
└─────────────────────────────────────────────┘
```

**样式特征**:
- **芯片式设计** (Chip/Tag style)
- 水平排列，间距 8-12px
- 选中状态: 蓝色勾选标记 ✓
- 未选中: 灰色边框
- 高度: ~32-36px
- 圆角: 6-8px
- 背景: 半透明或深色
- 悬停效果: 边框高亮

**当前实现的问题**:
- ❌ 位置错误：在 session header 下方，应该在输入框上方
- ❌ 样式错误：按钮式设计，应该是芯片式
- ❌ 交互错误：点击弹出模态框，应该是直接切换选中状态

---

### 3. Input Area (关键差异！)
**位置**: 主内容区底部

**官方设计**:
```
┌─────────────────────────────────────────────┐
│  Auto accept edits  ☐                       │  ← 左下角小字
├─────────────────────────────────────────────┤
│                                             │
│  Type / for commands                        │  ← 输入框
│                                             │
│                                      [Send] │  ← 右下角发送按钮
└─────────────────────────────────────────────┘
```

**样式特征**:
- **Auto accept 位置**: 输入框左下角外部（不是上方）
- **Auto accept 样式**: 
  - 小字体 (12-14px)
  - 复选框在右侧
  - 灰色文字
  - 悬停时高亮
- **输入框**:
  - 最小高度: ~80-100px
  - 圆角: 8-12px
  - 边框: 1px 灰色
  - 聚焦时: 边框变蓝色
  - Placeholder: "Type / for commands"
- **发送按钮**:
  - 位置: 输入框内部右下角
  - 样式: 圆形或圆角矩形
  - 颜色: 蓝色/橙色
  - 图标: 发送箭头

**当前实现的问题**:
- ❌ Auto accept 位置错误：在输入框上方，应该在左下角外部
- ❌ Auto accept 样式错误：按钮式，应该是文字+复选框
- ❌ 输入框高度不足：3 行太小，应该更高
- ✅ 发送按钮位置正确：在输入框内部右下角

---

### 4. Message Bubbles
**用户消息**:
- 背景: 深蓝色或深灰色
- 对齐: 右对齐
- 圆角: 12-16px
- 内边距: 12-16px
- 最大宽度: 70-80%

**助手消息**:
- 背景: 略浅的灰色
- 对齐: 左对齐
- 圆角: 12-16px
- 内边距: 12-16px
- 最大宽度: 100%（可以更宽）

**Tool Calls**:
- 嵌入在助手消息中
- 折叠/展开功能
- 图标 + 工具名称
- 灰色背景，圆角边框

## 🔄 交互模式

### Context Selector 交互
1. **点击芯片** → 切换选中状态（不打开模态框）
2. **选中状态** → 显示蓝色勾选标记
3. **未选中状态** → 灰色边框
4. **悬停** → 边框高亮

### Auto Accept 交互
1. **点击文字或复选框** → 切换状态
2. **选中状态** → 复选框打勾
3. **未选中状态** → 空复选框
4. **悬停** → 文字和复选框高亮

### 斜杠命令交互
1. **输入 `/`** → 显示命令面板
2. **命令面板**:
   - 位置: 输入框上方弹出
   - 样式: 深色背景，圆角卡片
   - 内容: 命令列表（图标 + 名称 + 描述）
   - 过滤: 根据输入实时过滤
3. **选择命令** → 插入到输入框
4. **ESC** → 关闭命令面板

## 📊 当前实现 vs 官方版本对比

| 组件 | 当前实现 | 官方版本 | 差异程度 |
|------|---------|---------|---------|
| **整体布局** | 三栏布局 ✓ | 三栏布局 | 🟢 匹配 |

| **输入框** | 3 行高度 | 更高（~100px） | 🟡 中等 |
| **消息气泡** | 基本样式 | 更精致的圆角和间距 | 🟡 中等 |
